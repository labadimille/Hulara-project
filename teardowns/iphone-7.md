# Teardown: iPhone 7

## Osservazioni Hardware
- **SoC:** A10 Fusion (CPU + GPU integrate).
- **Stato:** Dispositivo smontato per analisi architettura.
- **Difficoltà:** Alta (viti di lunghezza diversa, guarnizioni impermeabili).

## Obiettivi di studio
- Studiare il sistema di dissipazione del calore.
- Analizzare il bus di connessione del display OLED/LCD.
